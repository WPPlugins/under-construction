== Description ==

Displays a customizable 'under construction' page to visitors. A countdown can be displayed on the coming soon page as well as  links to twitter and other social networking sites. A email notifier form can be enabled or disabled.

== Installation ==

This section describes how to install the plugin and get it working.

 

1. Extract the archive and upload it to the plugin directory of your website.

1. Activate the plugin through the 'Plugins' menu in WordPress

1. Goto 'Under construction' under settings to customize the appearance and functionality.
